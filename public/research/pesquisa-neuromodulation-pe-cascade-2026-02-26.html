§2Conceito Fundamental

Neuromodulação

Definição: Neuromodulação é o processo pelo qual neurotransmissores específicos (dopamina, acetilcolina, serotonina, noradrenalina) alteram a atividade neuronal de forma difusa e prolongada, em contraste com a transmissão sináptica rápida e localizada. Neuromoduladores atuam como ganhos de controle em circuitos neurais, modificando a responsividade e a plasticidade sináptica.

Contexto histórico: A hipótese de que dopamina codifica reward prediction error (RPE) foi proposta por Schultz et al. (1997) e tornou-se um dos mapeamentos mais influentes entre neurociência e reinforcement learning. Doya (2002) expandiu esse framework, propondo que acetilcolina (ACh) controla o learning rate, noradrenalina (NA) modula exploration/exploitation, e serotonina (5-HT) afeta o discount factor.

Mapeamento Neuromoduladores → RL (Doya, 2002)
  • Dopamina (DA): Reward Prediction Error (RPE)
  • Acetilcolina (ACh): Learning Rate (α)
  • Noradrenalina (NA): Exploration (β, inverse temperature)
  • Serotonina (5-HT): Discount Factor (γ)

Prediction Error Cascade (PE Cascade)

Definição: A cascata de erro de predição é o processo pelo qual discrepâncias entre predições e entradas sensoriais propagam-se através de uma hierarquia cortical, com camadas superiores gerando predições top-down e camadas inferiores reportando erros bottom-up. Este framework é central à teoria de predictive coding (Rao & Ballard, 1999; Friston, 2005).

Evolução: Inicialmente proposto como modelo de processamento visual, o predictive coding expandiu-se para abranger percepção multissensorial, ação motora (via active inference), e mais recentemente, aprendizado hierárquico em redes neurais profundas (2020-2025).

§3Mecanismo

Neuromodulação: Como Funciona

Dopamina e Plasticidade Sináptica

Dopamina modula a plasticidade via ativação da cascata cAMP (adenilil ciclase). Experimentos de Brzosko et al. (2015, 2017) demonstraram que dopamina aplicada após a indução de STDP (Spike-Timing Dependent Plasticity) pode converter depressão sináptica (LTD) em potenciação (LTP), mesmo com delay de minutos. Isso implementa biologicamente a noção de eligibility trace em RL. 

Mecanismo de Neuromodulação Sequencial (Brzosko et al., 2017):

1. Exploração (ACh alta):
   pre → post pairing + ACh → t-LTD (depressão sináptica)
   
2. Reward (DA):
   t-LTD + DA (delay 1-5 min) → t-LTP (conversão para potenciação)
   
3. Resultado:
   Synapses ativas durante exploration + reward → potenciadas
   Synapses ativas sem reward → deprimidas
   
Equação:
Δw = η · A · STDP_window · eligibility_trace

onde:
- A = { -1 (ACh, sem DA), +1 (DA) }
- eligibility_trace = e^(-t/τ), τ ≈ 2s

Acetilcolina: Ativa receptores muscarínicos (M1), modulando a polaridade do STDP. Em presença de ACh, pairings pré-antes-pós induzem LTD (em vez de LTP), criando um viés para synaptic depression durante exploração. Isso permite unlearning de associações que não levam a reward.

Prediction Error Cascade: Componentes

Hierarquia cortical: O córtex cerebral organiza-se em níveis hierárquicos, com camadas superiores representando abstrações de maior nível e camadas inferiores processando detalhes sensoriais. Cada nível gera predições sobre o nível abaixo e recebe erros de predição de volta. 

Predictive Coding Hierárquico (Rao-Ballard Model):

Layer N+1 (abstrato)       ┌─────────────┐
                           │ Prediction  │
                           │    (P_N)    │
                           └──────┬──────┘
                                  │ (top-down)
                                  ↓
Layer N (intermediário)    ┌─────────────┐      ┌──────────┐
                           │   Input     │ ───→ │  Error   │
                           │   (I_N)     │      │  (E_N)   │
                           └─────────────┘      └────┬─────┘
                                                     │ (bottom-up)
                                                     ↓
Layer N-1 (sensorial)      ┌─────────────┐
                           │ Sensory In  │
                           └─────────────┘

Error computation:
E_N = I_N - P_N

Atualização de predição:
P_N ← P_N + α · E_N

Condições de contorno: O predictive coding assume que o sistema minimiza free energy (variational inference). Funciona bem quando o ambiente é estacionário ou semi-estacionário. Falha quando mudanças contextuais são radicais (requer reset via neuromodulação ou attention).

§4Estado da Arte

Neuromodulação em AI (2024-2026)

Lifelong RL via Neuromodulation (Lee et al., 2024): Framework que integra incertezas aleatórias (aleatoric) e epistêmicas para modular exploration e learning rate de forma adaptativa. Demonstrou navegação eficaz em multi-armed bandits não-estacionários sem hiperparâmetros fixos.

Avanços Recentes
  • Predictive Networks (PCNs): Implementações de predictive coding em PyTorch/JAX para deep learning, com performance competitiva a backprop em tarefas supervisionadas (2023-2025).
  • Active Inference Controllers: Controle robótico via minimização de free energy com neuromodulação dopaminérgica para seleção de políticas (2024).
  • Dendritic Error Computation: Modelos que implementam erro de predição em dendritos neuronais, resolvendo o problema de implementação biológica do predictive coding (Millidge et al., 2022).

Métricas: PCNs alcançam 97.8% no MNIST (vs 98.2% backprop), mas com gradientes locais e biologicamente plausíveis. Em tarefas de continual learning, neuromodulação reduz catastrophic forgetting em ~40% comparado a métodos baseline.

§5Debates Existentes

Predictive Coding: Teoria ou Metáfora?

Controvérsia central: Críticos argumentam que predictive coding é mais uma framework conceitual do que uma teoria testável com predições falsificáveis. Taatgen (2022) e outros apontam que muitos modelos de predictive processing podem gerar fenômenos empiricamente impossíveis ou ter contraexemplos não-capturados.

Debate sobre oscilações: A teoria padrão de predictive coding assume que sinais forward (bottom-up) e backward (top-down) operam em diferentes frequências oscilatórias (gamma vs beta/alpha). Evidências de fMRI e EEG são mistas — alguns estudos confirmam (van Kerkoerle et al., 2014), outros não encontram separação clara de frequências.

Neuromodulação: One-to-One Mapping?

Simplicidade vs Realidade: O mapeamento de Doya (DA→RPE, ACh→α, etc.) é elegante mas excessivamente simplificado. Dopamina, por exemplo, tem funções em movimento (Parkinson's), motivação (reward), e mais. Lerner et al. (2021) demonstraram diversidade de subpopulações dopaminérgicas com sinais distintos. A teoria unitária de DA como RPE está sob revisão.

§6Pontos de Crítica

Limitações Reconhecidas

Neuromodulação
  • Timescales mismatch: Neuromoduladores operam em escalas de tempo de segundos a minutos, enquanto comportamento requer decisões em milissegundos. Como o sistema resolve essa discrepância? (Frémaux et al., 2013)
  • Credit assignment problem: Em tarefas complexas, qual sinapse deve ser modificada após reward? Eligibility traces resolvem parcialmente, mas não completamente. [A] Brzosko et al., 2017
  • Biological plausibility: Modelos de neuromodulação em AI assumem sinais globais uniformes, mas biologicamente, release é local e heterogêneo. [A] Avery & Krichmar, 2017
Predictive Coding
  • Implementação com spikes: Teoria padrão usa unidades de rate-coding, mas neurônios reais são spiking. Implementar predictive coding com spiking neurons é desafiador (Denève & Machens, 2016). [T]
  • Precision weighting: Teoria requer que o sistema estime precisões (inverso de variâncias) de predições e erros. Como o cérebro computa isso? Mecanismo neural não está claro. [A] Feldman & Friston, 2010

§7Intersecção com Anima/Kerberos

Paralelos Concretos

O Anima implementa neuromoduladores (dopamine, acetylcholine) como sinais de controle que modulam adaptive gating entre hemisférios. O PE Cascade (L3→L4 resolution) funciona como hierarquia de predictive coding, com L4 (stakes detection) representando o nível abstrato e L3 (escolhas) o nível de ação.

O que podemos implementar diretamente:

  1. Eligibility Traces via Neuromodulação: Usar dopamina para converter depressões sinápticas em potenciações após reward, com decay exponencial (τ ≈ 2s). Já implementado no Anima v2.
  2. ACh-driven Exploration: Durante exploração (sem reward claro), ACh alta → bias para synaptic depression → permite unlearning de estratégias subótimas. Proposta: Implementar modo de "exploration purge" que deprime pesos quando confidence < threshold.
  3. Hierarchical PE Propagation: L3 gera predições sobre ações, L4 compara com stakes reais, erro propaga back para L3. Implementar como: error_L3 = stakes_detected - stakes_predicted, update weights com α * error_L3.

Proposta Acionável: Adaptive Learning Rate via ACh

Implementação no Anima:

1. Calcular incerteza epistêmica:
   epistemic_uncertainty = variance_across_ensemble(Q_estimates)
   
2. Modular ACh:
   ACh_level = sigmoid(epistemic_uncertainty / threshold)
   
3. Ajustar learning rate:
   α_t = α_base * (1 + ACh_level)
   
4. Durante alta incerteza:
   - ACh alta → learning rate alto → rápida adaptação
   - Durante baixa incerteza:
   - ACh baixa → learning rate baixo → consolidação
   
Código (pseudo):
if epistemic_uncertainty > threshold:
    learning_rate *= (1 + ach_gain)
    exploration_bonus += ach_exploration_boost

Benefício: Sistema adapta-se rapidamente a mudanças contextuais (high ACh) mas mantém estabilidade em ambientes conhecidos (low ACh). Isso alinha com a função biológica da acetilcolina em sinalizar unexpected uncertainty (Yu & Dayan, 2005).
